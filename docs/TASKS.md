# ✅ DailyFit 작업 목록 (TASKS.md)

## 상태 규칙

- [todo] 아직 시작 안 함
- [doing] 현재 작업 중
- [done] 완료

## MVP 단계

- **MVP v1:** T-001 ~ T-010, T-012, T-013 (출시 버전)
- **MVP v1.5:** T-011 (커뮤니티 확장)

## 관리 원칙 (Git Branch)

- 모든 개발은 `main` 브랜치에서 직접 수행하지 않고 개별 `feature/T-XXX-추가설명` 브랜치에서 작업합니다.
- 작업 완료(사용자 최종 승인) 시 `main` 브랜치에 Merge 하고, 작업 상태를 `[done]`으로 표시합니다.
